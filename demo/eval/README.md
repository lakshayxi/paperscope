# Demo evaluation fixtures (Phase 4B)

**Everything in this directory is synthetic.** `corpus.jsonl` is hand-authored fixture
data (fictional papers/reviews), not a real fetched OpenReview corpus. The predictions
in `generic_predictions.json` / `paperscope_predictions.json` / `baseline_predictions.json`
are hand-authored fixture values (each `run.settings.note` field says so explicitly), not
the output of any real model call. **`results/evaluation_report.md` and
`evaluation_results.json` are not a real benchmark result** — they exist only to show
what `paperscope evaluate`'s output looks like end to end.

Generated by running the real (offline, no-network) Phase 4B pipeline against this
fixture data:

```bash
paperscope prepare-eval --corpus demo/eval/corpus.jsonl \
                         --calibration-forums demo/eval/calibration_forums.json \
                         --output demo/eval/dataset --seed 42

paperscope validate-eval --dataset demo/eval/dataset \
                          --generic-predictions demo/eval/generic_predictions.json \
                          --paperscope-predictions demo/eval/paperscope_predictions.json

paperscope evaluate --dataset demo/eval/dataset \
                     --generic-predictions demo/eval/generic_predictions.json \
                     --paperscope-predictions demo/eval/paperscope_predictions.json \
                     --baseline-predictions demo/eval/baseline_predictions.json \
                     --output demo/eval/results
```

| File | What it is |
|---|---|
| `corpus.jsonl` | 12 synthetic forums: 4 calibration, 8 evaluation, spanning accept/reject/withdrawn/unknown decisions and two venue families |
| `calibration_forums.json` | The frozen calibration forum-ID list |
| `dataset/` | `prepare-eval`'s output: `model_inputs.jsonl`, `private_labels.jsonl`, `split_manifest.json`, `evaluation_manifest.json` |
| `generic_predictions.json` | Fixture predictions for an uncalibrated "generic" system (deliberately noisier, accept-skewed) |
| `paperscope_predictions.json` | Fixture predictions for a "paperscope" system (deliberately closer to the true labels, with calibrated-looking probabilities) |
| `baseline_predictions.json` | The reviewer-aggregate constant baseline, computed from the 4 calibration forums only |
| `results/` | `evaluate`'s output: `evaluation_results.json` + `evaluation_report.md` |
